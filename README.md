# Tousif Ahmed Portfolio

This repository contains a static portfolio webpage for Tousif Ahmed.

## Publish with GitHub Pages

1. Create a GitHub repository, for example: `tousif-ahmed-portfolio`.
2. Add the remote:
   ```powershell
   git remote add origin https://github.com/<your-username>/tousif-ahmed-portfolio.git
   git branch -M main
   git push -u origin main
   ```
3. In GitHub, go to Settings → Pages and set the source to `main` branch and `/ (root)` folder.
4. Your site will be available at `https://<your-username>.github.io/tousif-ahmed-portfolio/`.

## Notes

- `index.html` is the publish entrypoint.
- The site is fully static and requires no backend.
- Update `index.html` or `tousif-ahmed-portfolio.html` as needed.
